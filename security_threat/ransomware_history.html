                        # ランサムウェアの歴史

                        # ランサムウェアとは

                        コンピュータやファイルを使用不可能な状態に、回復と引き換えに身代金を要求するコンピュータウイルスである。 身代金(Ransom) を要求する マルウェア(Malware) なので、Ransomware である。


                        # 世界初のランサムウェア

                        ランサムウェアを先述の機能を持ったウイルスだと考えれば、観測上、世界初のランサムウェアは1989年に誕生していることになる。


                        要素 詳細
                        ウイルス名 AIDS Trojan
                        誕生日 1989年(平成元年)
                        感染経路 フロッピーディスクからインストール
                        身代金の金額 $189〜378(当時3〜5万円程度)
                        身代金の支払方法 犯人の私書箱に送金
                        標的 エイズ国際学会の出席者
                        人質 Cドライブ暗号化

                        1989年、パソコンにソフトウェアをインストールする方法はフロッピーディスクが主流であった。

                        突如、エイズ国際学会の出席者宛に送られてきたフロッピーディスクのラベルには「エイズ・ウイルス情報入門」と書かれていた。フロッピーディスクをパソコンに差し込むと、エイズ感染症に関する質問が表示される、というものであった。

                        しかし質問の裏では、こっそり別のプログラムがインストールされていた。 インストール後はPCの起動回数をカウントするだけで何もせず、起動回数が90回程度を越えたとき、突如Cドライブの暗号化を始めるという動きをした。暗号化処理が終わると、PC画面には送金を支持する内容が表示された。

                        しかし、支払先の情報から犯人が割り出され、被害が出ることはなかった。


                        # 支払い方法の模索

                        世界初のランサムウェア以降、何年も次のランサムウェアは観測されなかった。同様の行為をすれば簡単に逮捕されるからだろう。
                        その後、1998年にe-GOLDという仮想通貨が登場した。e-GOLDは通貨口座を作るとき、名前や住所が架空の情報 でも口座を作ることができたため、サイバー攻撃にぴったりだった。


                        要素 詳細
                        ウイルス名 GP Coder
                        誕生日 2005年
                        感染経路 ばらまきメールにウィルス添付
                        身代金の金額 $100〜200(当時1〜2万円程度)
                        身代金の支払方法 仮想通貨e-GOLD
                        標的 不特定多数
                        人質 ファイルの暗号化

                        しかし、e-GOLDはサイバー犯罪に共謀していると判断され、2009年に米国連邦大陪審によって業務停止が明示され、取引停止となった。

                        (参考)ドコモ口座事件

                        日本国内でも本人確認なしに口座を作れてしまう問題はあった。(2020年)

                        「ドコモ口座」事件の教訓はこれからのキャッシュレス業界をどう変えるか (opens new window) / ITmedia


                        # 心理的手法の活用

                        2011年、プリペイドサービスの金券を要求するランサムウェアが誕生した。 ランサムウェアをビジネスモデルとして確立するための試行錯誤が行われているように見える。


                        要素 詳細
                        ウイルス名 Reveton
                        誕生日 2011年
                        感染経路 ばらまきメールにウィルス添付
                        身代金の金額 $200(当時1万6千円程度)
                        身代金の支払方法 プリペイド(MoneyPak)
                        標的 不特定多数
                        人質 Windows起動ファイルの破壊

                        Revetonの特徴として、感染者に「支払わなくては」と思わせるために心理的手法が取り入れられた点が挙げられる。 感染すると画面にはFBIを名乗る警告文(あなたのパソコンが犯罪に使われており罰金の支払いが必要の旨)が表示される。 加えて、自身のIPアドレスを表示したり、パソコン内蔵のWebカメラの映像を画面に表示して まさに現在監視されている と思わせるようなウイルスである。


                        なお、2011年当時の為替(ドル円)は 1$=80円程度である。


                        # BitCoinの登場

                        2009年、BitCoinが誕生した。ここからしばらく、身代金要求のデファクトスタンダートが仮想通貨のBitCoinとなる。 BitCoinは個人が特定されることなく、国を超えた取引に手数料も掛からない。 大量の取引を重ねて資金源を追えなくするマネーロンダリングも容易である。


                        要素 詳細
                        ウイルス名 Crypt Locker
                        誕生日 2013年
                        感染経路 ばらまきメールにウィルス添付
                        身代金の金額 5BTC(当時6千円〜50万円程度)
                        身代金の支払方法 BitCoin
                        標的 不特定多数
                        人質 ファイル暗号化

                        暗号化強度も高く、プログラムの難読化やTor(ダークウェブ等に使われるブラウザ)を使った通信等、ランサムウェアとしての完成度が高い。
                        BitCoinを活用したCrypt Lockerの登場が、今後のデジタル身代金ビジネスを加速させたと考えている。


                        なお、2013円はBitCoinが世界的に注目を浴び、価格が劇的に変動した年である。


                        # Windows以外の端末を狙うランサムウェア

                        攻撃対象はWindowsだけではない。 スマートフォン世界シェア1位のAndroidを狙ったランサムウェアも登場した。


                        要素 詳細
                        ウイルス名 SimpLocker
                        誕生日 2014年
                        感染経路 ばらまきメールにウィルス添付
                        身代金の金額 $300(当時30万円程度)
                        身代金の支払方法 プリペイド(MoneyPak)
                        標的 不特定多数
                        人質 金銭要求メッセージの無限ループ

                        ひたすら 画面をロックした というメッセージを出し続けるものである。 ファイルの暗号化ではないものの、Android端末の操作ができなくなる。


                        # 新たな感染経路の模索

                        これまで、ターゲットにたどり着くために大量のメールを送信し、開封した人だけが感染するものであった。 感染させるための仕組みを作っておいて、ターゲットが勝手に感染する仕組みがあれば、攻撃者の負担は減る。

                        要素 詳細
                        ウイルス名 TaslaCrypt
                        誕生日 2015年
                        感染経路 不正ウェブサイト閲覧
                        身代金の金額 1.4BTC(当時3万円程度)
                        なお、時間内に払わないと倍額要求
                        身代金の支払方法 BitCoin
                        標的 不特定多数
                        人質 ファイル暗号化

                        不正なウェブサイトを閲覧するだけで、ウイルス配信サーバとの通信が始まり感染する仕組みである。 攻撃者は正規サイトを改ざんし、ランサムウェア配信サイトに書き換えていた。 普段からそのサイトを使っていた人は、知らぬ間に感染することになる。


                        なお、暗号化されたファイルの拡張子が.vvvになることから、vvvウイルス とも呼ばれる。


                        # WannnaCryの大流行

                        コンピュータウイルス自体が、自分のプログラムを他のコピーして拡大するワームと呼ばれる機能は以前より存在した。 2017年、そのワームが持つ特徴がランサムウェアに組み込まれ、世界的に感染が広がった。


                        要素 詳細
                        ウイルス名 WannaCry
                        誕生日 2017年
                        感染経路 ワーム型(自己増殖)
                        身代金の金額 $300分のBitCoin
                        なお、時間内に払わないと倍額要求
                        身代金の支払方法 BitCoin
                        標的 不特定多数
                        人質 ファイル暗号化

                        WannaCryの感染は世界150カ国、230万台もの端末が被害を受けたと言われている。 身代金を払っても暗号化から解消されなかった という報告もある。


                        # 標的型・脅迫型ランサムウェアの登場

                        データを暗号化するだけではなく、盗み取り、更にリークサイト(情報を暴露するサイト)で公開することを脅すランサムウェアが登場した。 ターゲットは個人ではなく、明確に企業であり、身代金の額も高額となる。


                        要素 詳細
                        ウイルス名 Maze
                        誕生日 2019年
                        感染経路 標的型メール
                        身代金の金額 850BTC(当時6億8千万円程度)
                        身代金の支払方法 BitCoin
                        標的 大企業
                        人質 データ暗号化、機密情報

                        データを盗み取ったことを証明するかのように、機密情報の一部がリークサイトに公開される。 あわせて、支払いが確認できなければ盗み取ったデータを全て公開する旨の脅迫がある。

                        ある企業では、Maze感染に対してセキュリティチームが復旧作業を行っていた。 その作業中のファイルすら、リークサイトに公開されたという。 一度標的になったら、徹底的に追い詰めるられる。


                        # まとめ

                        ランサムウェアは年々進化を重ねてきた。今後も更なる進化を続けるだろう。

                        要素 ランサムウェアの目指す方向
                        支払い方法 より犯人を特定できない方法へ
                        ターゲット より明確なターゲットへ
                        人質 より過激な脅迫方法へ
