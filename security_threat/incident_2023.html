# インシデント事例集(国内)

  • 上の方が新しい情報である。
  • なるべく公式報告の原文を引用するよう心がけている。
  • 魚拓(WEBのPDF)は取らない。著作権法違反の恐れがあるため。
  • 情報漏えい事故を探す時は「✔️情報漏えい」を検索。他に「✔️不正アクセス」、「✔️マルウェア」、「✔️サービス妨害」

# 1/5 株式会社メディウェル

Last update: 2023/1/9

確認項目 詳細内容
組織名 株式会社メディウェル
発表日 2023/1/5
侵害区分 ✔︎情報漏えい
✔︎不正アクセス
概要 1. 運営する「病院事務職転職ドットコム」の顧客情報データベースを標的としたサイバー攻撃
2. お客様の個人情報が流出している可能性
3. 以下の情報漏洩の可能性
-「医師転職ドットコム」又は「医師バイトドットコム」にてマイページをご利用のお客様と、「メディウェルログ」又は「クラヴィス（既に廃刊）」のいずれかのサービスにご登録いただいておりましたお客様の情報(合計3,325名分)
原因、経路 1. 開設日の2021年10月1日よりセキュリティホールが存在
2. 第三者による不正アクセス
暫定対応 1.セキュリティホールの修正
2.サイトの一時閉鎖
3.個人情報保護委員会へ報告
4.管轄の警察署へ通報
5.漏洩対象者に郵送またはメールにて順次個別の通知
今後の対策 1.第三者機関にて当社セキュリティ体制の調査及び改善
2.委託先の選定・監督の徹底
3.全てのサイトおよびネットワークに対する監視体制
4.第三者機関の調査結果に基づき、再発防止策を実施
5.WAF導入

# 1/4 ナッシュ株式会社

Last update: 2023/1/9

確認項目 詳細内容
組織名 ナッシュ株式会社
発表日 2023/1/4
侵害区分 ✔︎情報漏えい
✔︎マルウェア
概要 1.社内一部システムにおいて障害が発生
2.ランサムウェアによりPCのデータが暗号化
3.情報漏洩の可能性
- 個人・法人顧客情報（6,184件）
原因、経路 サーバーの脆弱性を利用した攻撃(のネットワーク設定やパスワードの強度の問題)
暫定対応 1.PCのネットワークを遮断
2.情報流出の可能性がある方々には、個別にメールにてご連絡
3.捜査機関、個人情報保護委員会及び顧問弁護士とも連携
今後の対策 再発防止に向けたセキュリティの強化等の対応をして継続
