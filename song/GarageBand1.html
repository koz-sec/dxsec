                  # 曲作りの動機(目的)

                  せっかく セキュアなICSクラウド導入指南書 (opens new window) を作ったが、正直あまり知られていない。
                  そこで、この指南書をモチーフにした曲を作って、普及活動を行おうと考えた。


                  文量も多くそれなりに難しい内容なので、読み込むにはそれなりに時間がかかる。
                  簡素化して、エッセンスだけでも多くの人に届けられたら、指南書を作った甲斐がある。


                  # GarageBandとは

                  作曲、録音、編集といった音楽制作を行うソフトウェアを DAW(Digital Audio Workstation) という。 GarageBandは、Mac OSに標準でインストールされているDAWである。


                  # 起動する

                  Garage Band を起動し、空のプロジェクト を選択する。 トラックのタイプを聞かれるので ソフトウェア音源 を選択する。


                  # 音符入力の準備

                  初期起動時はピアノ音源が選ばれている。 トラック上で Command を押しながらドラッグすると、音程を入力可能な緑色のボックスができる。


                  この緑色のボックスをダブルクリックすると、画面下にピアノロールが表示される。
                  縦軸が音の高さ(上にいくほど高い音、下に行くほど低い音)であり、横軸は時間である。 譜面と似ている。


                  Command を押しながらクリックすると、音符を入力できる。


                  # ドラムトラックの作成

                  GarageBand に限らず、DAW は楽器等のパートごとにトラックを分けて入力する。
                  ピアノトラックの下にある空白部分を右クリックするとメニューが現れる。
                  新規ソフトウェア音源トラック を選択すると、新しいトラックが追加される。


                  画面左に楽器を選ぶ欄があるので、Drum kit から好きなものを選ぶ。 ここでは、Smashというドラムを選択した。

                  なお、インストールされていない楽器は薄い灰色で表示される。クリックすればインストールが始まる。


                  # 仮ドラムの打ち込み

                  ドラムは、1つの音程に1つの部位(シンバル、ハイハット、スネア、バス等)が紐づいている。 どの音程がどの部位に紐づいているかは、使用するドラムセットによって異なるため、試行錯誤で探す。

                  Smashドラムの紐付け

                  例えば、私が選択したSmashであれば、G#(ソ#のこと)のC1(下から1番目のオクターブ)がクローズドハイハットに該当する。 B0(0番目のオクターブ内のシ)がバスドラム、D1がスネアの音だった。

                  # メロディ入力

                  曲を作る手順は人それぞれの好みで良いと思う。 以前の私はコードから作っていたし、歌詞を先に作る人も多いと思う。 今回私は、サビのメロディを先に作って、思いつきで他のメロディにつなげていく。


                  今回サビの歌詞にどうしても持ってきたいキーワードが「ICSクラウド」である。
                  当初の目論見である セキュアなICSクラウド導入指南書 (opens new window) の普及 を効果的に行うには、最も伝えたいキーワードがサビで使われるようにしたい。 つまり「I・C・S　クラウド〜♪」といった歌詞と共にサビが始まれば、最もインパクトがあると考えた。
