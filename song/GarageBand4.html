        # 前回の結果

        前回、ドラムやベースの配置を変更し、曲らしくなってきた。


        Guitarの音がチープで、打ち込み感が残っている。

        本格的なGuitarの音に変更する。


        # ギター音の追求

        Standard Guitarというサウンドフォントを使用する。

        音原作者：紫式部🐰 @Purple_Shikibu_ (opens new window)さん


        音のクオリティも高いが、標準の打ち込みでは再現困難なギター奏法(ミュート、ハーモニクス、ピックスクラッチ等)を実装できる。素晴らしい..

        # 必要なソフトウェアのインストール

        # Standard Guiterのダウンロード

        まずはStandard Guitarをダウンロードする。

        • Unreal Instruments (opens new window)を開く
        • 画面下にスクロールして、Free Downloadを押す。
          • 細かい手順はOSによって異なるので省略する。
        • ダウンロードしたファイルを解凍する

        # sforzandoのインストール

        音源のStandard GuitarはそのままではGarageBandで読み込めない。 読み込ませるために、sforzando(以下、sfz)というソフトウェアを使う。

        Download はこちら → sfz (opens new window)

        • リンクを開いて、中央あたりにあるDownload for freeを押す。
        • OSを選ぶ画面に遷移するので、自分のOSを選ぶ(私はMacOS)
        • ダウンロードしたファイルを実行して、インストールする
          • 細かい手順はOSによって異なるので省略する

        # sforzandoとStandard Guitarの接続

        • sfzを起動する
        • sfzのウィンドウに、Standard Guitarのxmlファイルをドラッグする

        # GarageBandでの使用

        • インストールした後にGarageBandを起動すると、トラック設定でsfzが選べるようになる
        • INSTURMENTでStandard Guitarが選べるようになる。

        # アンプ設定

        このままだと、クリーンギターの音しか出ない。 ロックな音(歪んだ音)を鳴らすためには仮想アンプが必要になる。 初期設定は大変かもしれないが、様々なアンプと組み合わせられる点は、Standard Guitarの良いところである。

        # BlueCat's FreeAmpのインストール

        • BlueCatAudio (opens new window)を開く
        • いくつか似た名前のソフトウェアが並ぶが、その中からBlue Cat's Free Amp(以下、BCamp)を探してダウンロードする。
          • わからない人は、こちら (opens new window)がダウンロードページ
        • インストールする
          • 細かい手順はOSによって異なるので省略する

        # BCampの使用

        • トラック設定のAudio Unitsに表示されるBCampを選択する。
        • 好きなアンプを選ぶ。好きに設定する。

        ここまで設定すれば、ギターっぽさが出る。
