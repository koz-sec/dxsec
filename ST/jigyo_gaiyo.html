        # 事業概要、事業特性

        120分で約3,000文字を書き切るには、論述する企業やDX事例のストーリーをあらかじめ考えておかなければ難しい。仮に実務経験を元に論述する場合でも情報整理しておくと良い。


        私は普段の業務で直接的にDXと携わっていないため、身近な企業をベースにした 架空のDXサクセスストーリー を作って試験対策とする。


        サクセスストーリーに至る流れは以下のイメージ。従って、本章では欠陥のある企業を描いていく。

        1. DXに取り組む体力がある企業が存在
        2. 事業環境が変化
        3. うまく対応できず、いくつかの欠陥がある状態
        4. DXの取り組みによって欠陥を解消
        5. 大　成　功

        ここで、成功を判断する指標は「利益」ではなく「売上」とする。
        仮に「利益」を指標とした場合、売上拡大と費用削減の両面で論じることとなる。 すると複雑な論述になってしまい、文章構成が難しくなってしまうためである。


        # 参考にする資料

        売上を伸長させるDX(いわゆる攻めのDX)に関するサクセスストーリーを描く場合、BtoB事業より、BtoC事業の方が良い。 一般消費者はさまざまなニーズを有しており、論述のネタはいくらでも存在するためである。

        BtoC事業をベースに論述構成を検討する。


        少し前に読んだ アパレル・サバイバル (opens new window) から良いとこ取りして、アパレル業界のDXを論述してみようと思う。
        あと、マネーの拳 (opens new window) もアパレル業界の集中戦略や差別化を取り扱っていた。参考にする。

        # アパレル事業の特性

        • 内部環境

          • 同じ洋服でも、サイズ別かつカラー別でそれぞれの在庫管理が必要
          • 自社ならではのデザインとこだわりの素材で差別化

          • ファストファッションが広まり、差別化が必要

          • 顧客ニーズが多様かつ流行により商品ライフサイクルが短い

        ※ファストファッションも、新型コロナの影響で撤退が相次いでいる。ここでは考慮しない。

        # モジュール化

        • 事業の概要

        A社は、自社で製造したアパレル製品を販売する小売業を営んでいる。企画、デザイン、製造および販売までを一貫して行うSPA企業である。

        本社でデザインしたアパレル製品を、中国およびカンボジアの子会社で製造し、国内12箇所の店舗で販売を行なっている。 A社が販売するアパレル製品は、シンプルなデザインに定評があり、働く女性からの支持が高い。


        • 事業の特性

        アパレル業界は参入障壁が低く、価格競争に陥りやすいという特性がある。

        新型コロナウィルス感染症の拡大による来店客の減少や、テレワークの浸透による需要減により、売上は急激に減少した。
