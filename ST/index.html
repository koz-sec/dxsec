# 午後1対策

与件文に対し、数十文字程度で論述する試験である。 与件に寄り添い、書かれている言葉を引用する形で答えていけばよい。

解答の記述方法にコツが要るので、決まった様式に慣れる対策が必要。


# 午後2対策

近年はデジタルトランスフォーメーションに関する問題が出題される。DXの取り組みにあたり「どんな課題に直面したか」と「どう乗り越えたか」について論述することが求められる。
実際の所属企業のことを書く必要はない。架空企業のストーリーでも良い。 BtoC事業者をイメージして攻めのDXについて論述しても良いし、BtoB事業者前提で守りのDXを述べても良い。
