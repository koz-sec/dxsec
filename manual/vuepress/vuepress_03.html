          # VuePress - 03: アレンジ設定

          # 設定をアレンジするために

          • vuepressの設定ファイルは初期状態だとローカルにありません。eject を使って現在の設定をファイルに書き出すことで、自分好みに変更できるようになります。
          • 自分の好きなテーマカラーに変更したり、レイアウトをいじったりできそうです。

          # 設定を書き出す

          • packerage.json を編集モードにして、scriptsに eject を追加で書き込みます。
            "scripts": {
    ...
    "eject": "vuepress eject src"
    ...
  },
          • 以下のコマンドを実行します。
          yarn eject

[DONE] Copied default theme into /Users/koz/Desktop/VUEPRESS/src/.vuepress/theme

          • どうやら、実行したフォルダのすぐ下に src フォルダができてしまったようです。ココジャナイ。

          • どういうファイルが書き出されたのか見て見ましょう。

          ls src/.vuepress/theme

AlgoliaSearchBox.vue	Navbar.vue		SidebarGroup.vue
DropdownLink.vue	NotFound.vue		SidebarLink.vue
DropdownTransition.vue	Page.vue		search.svg
Home.vue		SWUpdatePopup.vue	styles
Layout.vue		SearchBox.vue		util.js
NavLink.vue		Sidebar.vue
NavLinks.vue		SidebarButton.vue
          • これらのファイルを、docs/.vuepress/theme に移動すると反映されるようになります。

          # テーマカラーを変更する

          デフォルトは黄緑です。このままだと VuePress感が強くて、既製品をそのまま使ってるーって思われそうな気がします。私はユニクロの服が好きだけど、ユニクロ着てるなって分かっちゃいますよね。そんな感じです。

          • docs/.vuepress/theme/style/platte.styl ファイルを作成します。
          • ファイルの中身は一旦デフォルト値で埋めましょう。
            • 以下はデフォルト値
          $textColor = #2c3e50
$borderColor = #eaecef
$codeBgColor = #282c34
$arrowBgColor = #ccc
$badgeTipColor = #42b983
$badgeWarningColor = darken(#ffe564, 35%)
$badgeErrorColor = #DA5961

// layout
$navbarHeight = 3.6rem
$sidebarWidth = 20rem
$contentWidth = 740px
$homePageWidth = 960px

// responsive breakpoints
$MQNarrow = 959px
$MQMobile = 719px
$MQMobileNarrow = 419px

// accent color
$accentColor = #3eaf7c
          • これで、好きなように色やレイアウトを変更すると良いですね。
          • 私はあちこちブルー基調にしてみました。
            • #092635
            • 1B4242
            • 5C8374
            • 9EC8B9

          # 検索バーの位置を変更する

          • 一番右に検索バーがあった方が良い
          • Navbar.vue を編集します。結構下の方にスクロールすると、.search-boxがあります。
            .search-box
    margin-left 1rem //←この行を追加してください！
    flex: 0 0 auto
          • 検索ボックスが、画面の一番に移動しました。
