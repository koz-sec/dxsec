              # Twitter API(1)

              Twitter APIについては、長くなるので章を分けて記載する。

              • Twitter APIの利用申請

              # API利用の目的を決める

              API利用目的は、TwitterAPIを使ってみたいという単なる好奇心でも良い。 しかし、Twitterデータには個人情報が含まれることもあり、使用用途を決めて適切にデータ管理をすることが望ましい。

              私は、セキュリティ関連情報を収集し、ツイート機能で情報共有を行う。 という目的を設定した。
              このように全自動で情報発信を行うアカウントをBOTと呼ぶ。以降はBOTを作成する手順を記載する。


              # Twitterアカウントを作成する

              Twitterアカウントを作成するにはメールアドレスが必要である。

              1. Gmailのアカウントを作成 (opens new window) する。
              2. Twitterアカウントを作成 (opens new window) する。

              # TwitterAPI利用を申請する

              1. ブラウザ版Twitterでログインする
              2. Settings > Your Account > Account Information の順に開く
              3. Phoneを選択して電話番号を登録する

              電話番号登録の理由

              電話番号を登録していないとAPI登録申請時にエラーが出るため

              1. 開発者専用ポータル (opens new window)を開く

              2. 質問に答える

                質問 私の回答
                What country are you based in? Japan
                What's your use case? Making a bot
                Will you make Twitter content or derived information available to a government entity or a government affiliated entity? No
                Get involved and updated ✔︎checked

              3. 規約を読み、合意にチェックをつけてsubmit

              4. Gmail宛にメールが届くので、confirm your emailボタンを開く

              5. APIに名前をつける(他の人が既に使っている名前は使えない)

              6. APIキーが表示されるが、今は気にせずDashboardへ移動する

              APIの名前

              上記の手順8で設定したAPIの名前は、APIを介してツイートをした時に表示される。他の人に見られても問題のない名称をつける。


              # Twitter APIで接続する準備

              開発者専用ポータル (opens new window)に移動し、APIのアクセス権を設定する。

              • User authentication settings > Set up
              • App PermissionをRead and Writeに設定
              • Type of appをBotに設定
              • Redirect URL / Website URLは適用に入力
              • Client ID / Client Secretをコピーして保管 ※取扱注意
              • Key and Tokensを開く
              • Bearer Tokenは使わないのでRevoke
              • Access token/Secret Access TokenをRegenerate
              • 表示されたtoken / Secret tokenをコピーして保管 ※取扱注意

              要注意！

              この操作で表示される4つのAPIキーでアカウントを自由自在に操作することができます。他の人に見られない場所・方法で厳重保管してください。


              PCからAPIを使用するため、ターミナルでtweepyをインストールする

              pip3 install tweepy

              # APIで接続する

              試しにAPIでツイートする。テキストエディタ等に以下のコードを貼り付け、太字の部分を自身のAPIキー等に変更する。

              import tweepy

              consumer_key = 'client_key'
              consumer_secret = 'client_secret'
              access_token = 'access_token'
              access_token_secret = 'access_token_secret'

              api = tweepy.Client(
              consumer_key = consumer_key,
              consumer_secret = consumer_secret,
              access_token = access_token,
              access_token_secret = access_token_secret
              )
              api.create_tweet(text = "ここにツイートする内容")


              プロンプトを開き、python3を起動する。
              先ほどのコードを貼り付けると実行される。

              図: 自身のアカウントを見るとツイートされていることが分かる。


              なお、私はVisual Studio Codeを使ってPythonを書き、プログラム名.py で保存している。 保存すれば実行するだけで良い。
