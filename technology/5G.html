            # ローカル5G


            # スマートフォンの無線通信を支える基地局

            スマートフォンで使用される3G/4G回線は場所を気にせず使うことできる。 これは、基地局 という通信アンテナが日本中に設置されているためである。

            この基地局は主に、NTTドコモ、Softbank、KDDIの3社が保有している設備である。 加えて、近年では楽天が基地局を増やし、3大キャリアが4大キャリアになろうとしている。


            基地局は鉄塔、山の中、ビルの屋上等、あまり人目につかない場所に設置されている。
            安定した通信を最小限の設置台数(=低コスト)で実現する方法は、各キャリアが競争優位を維持するノウハウであるため、具体的な設置場所は公開されていない。


            図: 基地局の種類(引用元: 携帯電話基地局とわたしたちの暮らし (opens new window) / 総務省)


            # 自社で基地局を保有するローカル5G

            情報処理の高度化やIOT機器の登場もあり、事業者の多様なニーズが生まれた。

            ニーズの例

            • アップロードとダウンロードの速度をカスタマイズしたい
            • 自社だけの環境で機密性を高めたい
            • 大量の機器を接続したい

            そこで、自社敷地内に自社専用の基地局を設置し、自分好みにカスタマイズできる「ローカル5G」という制度が誕生した。
            使用者は自社従業員や自社のIOT機器だけなので機密性が高く、用途によってアップロードとダウンロードの比率を変更し、リアルタイムな映像転送も実現できる。


            # ローカル5Gを使う方法

            しかし、この制度はどの会社でも使えるわけではない。基地局は法律で「特定無線設備」と位置付けられており(電波法 第38条 2-2)、管理運用をする有資格者の設置や、総務省への届出が必要である。


            大まかな手続きの流れ

            • 設置場所の調査・検討を行う(他設備への電波影響調査)
            • 所定の書式で総務省に届け出る
            • 総務省が審査を行う(場合により質疑が発生)
            • 第三級陸上特殊無線技士 資格保有者が運用する

            また、ローカル5Gを使って営む事業内容によっては、国の許可(電気通信事業法 第9条or第16条)が必要となるケースもある。


            # ローカル5Gの検討状況

            ローカル5Gは、既にさまざまな分野で活用が検討されている。


            農業分野での事例:

            • 草刈・防除ロボットの遠隔監視
              項目 詳細
              課題 除草作業は、労働力不足や薬剤散布による健康被害の解消
              実験内容 草刈・除草ロボット(EV車)を走らせ、リアルタイム監視と制御と行った。
              実験結果 遠隔からの停止操作を検証し、1秒以内にロボットを停止できることが確認できた。

            工場での事例:

            • 遠隔での技術継承
              項目 詳細
              課題 高度な溶接技能を持つ人材不足、技術承継の遅れ
              実験内容 若手が行う作業の映像や電圧等の情報を集め、リアルタイムで熟練工に伝送し、遠隔指導を行った。
              実験結果 会議室からの遠隔指導により、溶接品質を1割から8割まで引き上げることに成功した。

            空港での事例:

            • 連絡バスでの映像伝送
              項目 詳細
              課題 空港機能の拡張と労働減少による将来的なドライバ不足
              実験内容 空港の連絡バスを活用した自動運転のため、高画質・高フレームレートの映像を伝送を行った。
              実験結果 遠隔操作、異常時のキャリア切り替え等に成功し、自動運転の発展に寄与した。

            この他にも、参考URLに記載した令和3年度ローカル5G開発実証報告書にて、さらに多くの検討事例が記載されている。
