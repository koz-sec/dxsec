            # VuePress(2) - OGP設定

            # OGPとは

            OGP(Open Graph Protocol)は、SNSでサイト紹介した時に表示されるバナーのことらしい。

            • 設定前
              単にアドレスだった

            • 設定後
              画像と説明文に変わった


            # OGPで表示する画像の準備

            表示したい画像を用意してみた。 Google Slideでオブジェクトを並べただけ。センスが良すぎて申し訳ない。


            # OGPの設定

            VuePressの設定としては、.vuepress/config.js に以下の内容を加筆すれば良い。

            module.exports = {
  head: [
    ['meta', { name: 'twitter:site', content: '@dxzukan' }],
    ['meta', { name: 'twitter:text:title', content: 'ぼくの全チ全ノウ -ビジネス+IT+セキュリティ-' }],
    ['meta', { name: 'twitter:image', content: 'https://koz-sec.github.io/dxsec/OGP_logo.png' }],
    ['meta', { name: 'twitter:image:alt', content: '経営とセキュリティの橋渡しをスムーズにするため、僕の持っている知識をどんどん書き溜めていきます。' }],
    ['meta', { name: 'twitter:card', content: 'summary' }]
    
  ]
}

            そして、作った画像は .vuepress/public に設置す流。


            設定が終わると、Twitter投稿時に表示されるようになる。(再掲)

            ちなみに、過去に投稿した分は投稿当時から変わらない。


            # OGPの変更

            画像とか説明文を変更してもTwitterへの反映は時間がかかる。 それは Twitter側のキャッシュ に古い情報が残っていて更新されていないから。

            以下のページで対象ページの表示検証を行うと、Twitter側のキャッシュも更新されるとのこと。

            https://cards-dev.twitter.com/validator

            エラー画面？

            Unable to render Card preview と表示されるけど、そういう仕組みらしい。


            うまく更新されたかどうかは、自分のアカウントでツイート投稿画面を開き、URLを入力すれば確認できる。
