              # Twitterの検索に失敗！

              下記は、いくつかのウェブサイトを拝見して作ったプログラムである。 目論みとしては、ランサムウェアというキーワードの含まれるTweetを取得するつもりだった。 しかし、このPythonプログラムを実行したところ、うまく取得できなかった。

              import tweepy

# 検索ワードを決めて、
searchword = 'ランサムウェア -filter:replies'

# 認証情報を入力して、
api_key = '私のAPIキー'
api_secret = '私のAPIキー'
access_token = '私のトークンキー'
access_token_secret = '私のトークンキー'

# Twitterで認証して
auth = tweepy.OAuthHandler(api_key, api_secret)
auth.set_access_token(access_token, access_token_secret)
api = tweepy.API(auth, wait_on_rate_limit = True)
    
# ツイートを取得する。
tweets = api.search_tweets(q=searchword, result_type='recent', count=20)
    
# その結果は？
print(tweets)

              結果として表示されたのが以下の内容である。

              tweepy.errors.Forbidden: 403 Forbidden 453 - You currently have Essential access which includes access to Twitter API v2 endpoints only. If you need access to this endpoint, you’ll need to apply for Elevated access via the Developer Portal. You can learn more here: https://developer.twitter.com/en/docs/twitter-api/getting-started/about-twitter-api#v2-access-level

              要するに下記の状況であると教えてくれている。

              • 現在の権限ではこの操作は禁止されている。
              • Elevated Access を取得する必要がある。

              # Twitter権限の種類

              Twitter APIには権限の種類がある。

              • Essential
                最初の申請を終えた段階ではEssential権限であり、複雑な操作はできない。

              • Elevated
                Elevated権限を得ることで、高度なAPI機能を使用することができる。

              • Elevated+, Managed
                非常に多くのデータを処理できたり、アカウント管理ができるが、有償である。

              # Elevated Access申請

              Developer Portal で自分のプロジェクトを開き、Elevated Access の申請を行う。 (Twitterは API v2 の使用を推奨しているが、それは後から考えることにする)


              Elevated Access の権限は、Twitter社の審査を通った場合に限り適用される。(拒否されることもある) 審査画面にすすむと、画面にいくつかの設問が表示されるので、下記の通り回答した。
