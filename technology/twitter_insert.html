            # Twitterタイムラインの埋め込み

            # ゴール

            Twitterタイムラインが埋め込まれブログは結構ある。 埋め込むと以下のようになる。

            ※読み込みに時間が掛かります。表示されるまでお待ちください。

            Tweets by koz_sec

            # 作成手順

            • https://publish.twitter.com/ (opens new window) にアクセスする。

            • 表示したいアカウントの名前を入力し、Enterを押す

            • Embedded Timeline を選択する

            • 画面に表示されるHTMLタグをコピーし、ブログに貼り付ける。


            HTMLタグをブログに貼り付けた結果は以下の通りである。 縦方向にとても長いため、詳細は畳んである。
            クリックで開く

            Tweets by koz_sec


            # 表示のカスタマイズ

            Twitterのつぶやきをブログに表示することができたものの、縦横のサイズが大きく使いにくい状態となっている。 使いやすいサイズにカスタマイズしていく。

            • set customization options を選択する

            • 以下のように入力する

            項目 入力する内容 項目の意味
            Height 250 表示領域の縦幅(px)
            Width 400 表示領域の横幅(px)
            How would you like this to look? Dark テーマカラー(Light or Dark)
            What language would you like to display this in? Japanese 表示言語

            • 入力した画面は以下のようになる。

            • 画面下にイメージが表示されるので、問題なければ Update を選択する

            • 画面に表示されるHTMLタグをコピーし、ブログに貼り付ける。
