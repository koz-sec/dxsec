                # 消費関数

                # 所得・消費・貯蓄の関係

                経済学において、消費者のお金の流れは非常にシンプルである。 所得から消費を除いた残りが貯蓄というシンプルな選択肢だけが用意されている。

                図: 3要素の関係から筆者作成


                # 消費の分類

                ケインズは、消費に性質があると考え、基礎消費 と 限界消費性向 に分類した。
                所得は、貯蓄、基礎消費、限界消費性向の3つに分類される。


                • 基礎消費(Consumption)
                  所得の大小に関係なく生活するために必要最低限の固定支出をいう。たとえば家賃等である。
                  大文字の「C」で表現される。

                • 限界消費性向(Propensity to consume)
                  所得の大小と比例して支出される変動支出をいう。買う余力があれば、生活水準を上げる。
                  小文字の「c」で表現される。

                増えた所得をなぜ貯金に回さないのか

                所得が大きければ気持ちも大きくなって消費してしまう、というのは現実世界の話でもありうる。
                経済学では、消費することが自信の利得 であることが前提となっているため、資金の余力はなるべく購買に使われる。


                (補足)

                • 所得(Yield)
                  産出量のことを表す。大文字の「Y」で表現される。経済学においては三面等価の原則により、産出量が国民所得と等しい。

                • 貯蓄(Saving)
                  銀行預金等、将来のために蓄えることを言う。大文字の「S」で表現される。


                # ケインズの(短期的)消費関数

                変動支出は所得に応じて変化する。 変動支出は、Y(所得)とc(限界消費性向)を掛け算した値となる。固定支出はC(基礎消費)である。
                この関係を数式にすると以下の通りである。

                C(総消費) = C0(ある時点の固定支出) + c0Y(ある時点の変動支出)


                # 貯蓄関数

                Y(所得)からC(総消費)を引いた残りがS(貯蓄)である。

                S(貯蓄) = Y0 - { C0(ある時点の固定支出) + c0Y(ある時点の変動支出) }


                これらの関係を図示すると下図の通りである。

                # クズネッツの(長期的)消費関数

                先述したケインズの消費関数では、所得が激減しても固定費を払い続けることが前提になっている。 現実に即した考え方をすれば、所得が減れば 基礎消費(例えば家賃等) を見直すため、基礎消費も変動的だと考えられる。

                クズネッツは、アメリカにおける1940年代から1950年代の実証データに基づいて、以下の公式を導き出した。

                C(消費関数) = 0.9Y(所得)

                所得の9割を消費し、1割を貯蓄に回すということが表現されている。 切片が0のため、原点を通るグラフとなる。


                # 2つの消費関数の議論

                ケインズの考えとクズネッツの考えが出てきた後、いくつかの仮説が提示されて議論された。 長い議論を経て、ケインズは短期消費関数、クズネッツは長期消費関数であると結論された。

                • 相対所得仮説
                  デューゼンベリーは、消費(C)は現在の所得(Y)だけでなく、過去の最高所得(Ym)にも依存すると考えた。 そのため、急激な所得変動があっても、消費は緩やかな変動をするという仮説を唱えた。

                  過去の輝かしい所得額が購買力に影響するため、たとえ急激に不景気となっても一時的には購買力が落ちない状態が続くと考えた。 これを ラチェット効果 と呼び、景気の急激な後退を歯止めする効果がある。

                • 恒常所得仮設
                  フリードマンは、所得は2種類に分類できると考えた。 固定的に得られる給与等の所得を恒常所得(Yp)と呼び、賞与等の一時的な所得を変動所得(Yt)とした。
                  消費(C)は恒常所得(Yp)に依存するため、一時的な収入があっても消費行動は変わらないと考えた。

                • ライフサイクル仮設
                  モジリアーニは、消費(C)は生涯かけて得られる総所得に依存すると考えた。 そして、資産は死ぬまでに残さず使うという前提を置いた。
                  将来得られる見込みの総収入を考慮すると、一時金では生活水準を上げることはなく、昇給等の長期的な収入が期待できるときに生活水準を上げる。
