                    # QC7つ道具

                    製造業にとって、Quality Control(品質管理)は欠かせない。 良い品質の製品を作るため、データを収集し、分析して、意思決定に活用する。
                    分析によく知られる手法として QC7つ道具 がある。


                    # QC7つ道具の概要

                    代表的な7つの分析ツールは下表のとおりである。

                    ツール名 見た目 使用目的
                    グラフ 図の総称(折れ線、レーダーチャート、円グラフ等) データを視覚的に表すため
                    パレート図 棒グラフ(多い順)に、折れ線(累積比率)を重ねた図 比率の多い要素を抽出するため
                    特性要因図 フィッシュボーン 事象を原因に分解して分析するため
                    ヒストグラム 棒グラフまたは折れ線グラフ 分布(ばらつき)を見つけるため
                    散布図 2主の測定値をプロット 相関関係を見つけるため
                    管理図 折れ線に上限(UCL)や下限(LCL) 異常値(品質不良)を見つけるため
                    チェックシート 確認項目のリスト 漏れや品質管理をするため

                    # グラフ

                    引用元: グラフの基礎知識 (opens new window) / FujiFILM

                    データを視覚的に表現することでわかりやすく表現したもの。 図で表現することで新旧対比や主張する内容が一目で分かるようになり、分析や意思決定に使いやすくなる。


                    # パレート図

                    引用元: パレート図（ぱれーとず） (opens new window)/ ITmedia

                    ビルフレッド・パレートが発見した法則を パレートの法則 という。 別名 8:2の法則 といい、売上の8割は2割の社員に依存するという経験則を述べた。
                    上図においては、問題の原因は「キズ・歪み・そり」の3項目が全体の8割を占めることが分かる。 そのため、どこを改善すると効果が高いかすぐに分かる。


                    # 特性要因図

                    引用元: 中小企業診断士 平成28年度 第2次試験問題 事例Ⅲ (opens new window)

                    1つの問題がどのように発生したのか分解して表現する。 上図では、加工不良が多い原因を 4Mフレームワーク(Man, Machine, Method, Material) に分けて分析を行なっている。


                    # ヒストグラム

                    引用元: ヒストグラム (opens new window) / なるほど統計学園

                    データをいくつかの階級に分類して、ばらつきを発見する。 上図(架空データ)では、160cm~165cmが最も多いことが分かり、175cmを超える人は非常に少ないことが分かる。

                    引用元: ヒストグラム (opens new window) / なるほど統計学園

                    グラフの形によってばらつきの特徴を読み取ることもできる。
                    多峰性(たほうせい)のグラフでは、ある地域において 40代の親が多ければ子供である10代も多い といった関係が見えることもある。


                    # 散布図

                    引用元: 散布図 (opens new window) / LightStone

                    2つのデータをブロットし、相関関係の有無を分析する。 右肩上がりならプラスの相関関係、右肩下がりならマイナスの相関関係になる。

                    相関係数

                    相関係数は -1 から 1 の値を取る。
                    1に近いほどプラスの相関があり、一方のデータが変化したとき同じ変化をする。 -1に近いほどマイナスの相関があり、一方のデータが変化したとき反対の変化をする。 相関が全く見られないとき、相関係数は0になる。


                    # 管理図

                    引用元: 「安定した工程」を作るには？ ゼロから学ぶ「管理図」の使い方 (opens new window) / ITmedia

                    品質特性をプロット(打点)することで品質の管理を行う。
                    管理図には種類があり、上図は C管理図 である。 例えば、品質特性を製品の硬度(かたさ)とする。硬すぎても、柔らかすぎても、製品として使い物にならない。 このような状況で管理図を使うには、上限(UCL: Upper Control Limit)と下限(LCL: Lower Control Limit)を設定し、品質異常を発見する。


                    その他の管理図

                    • X-R管理図
                      X(平均値)とR(範囲:Range)を合わせた管理図である。 Xは平均値を打点し、Rではばらつきの範囲を打点する。
                      平均だけ、ばらつきだけでは抽出できない品質のブレを検出できる。

                    • P管理図
                      不適合品の割合から、作業工程を評価する管理図である。 割合を使うため、生産個数にばらつきがあっても適用できる。
