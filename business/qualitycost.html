              # 品質不良

              ものづくりの現場では、製造に失敗し、手直しや廃棄が発生することがある。
              例えば、精密機械や医療機器等の高い品質が求められる産業分野では、わずかな歪みがあるだけで出荷不能になる。食品製造であれば、昨今の食品安全意識の高まりにおいて、異物混入は看過できない。

              品質不良の製品が顧客の手に渡った場合、損害賠償、信用問題、更には人命に関わる問題に発展することもある。


              # 品質不良に関する事例

              項目名 内容
              事例 不適合医療品の出荷
              参考URL 日医工の医薬品不正製造 (opens new window) / 富山県薬事審議会
              品目 医薬品
              発生時期 2020年

              ジェネリック薬品を製造販売する日医工は、医薬品の品質や安全に関する法律である 薬機法 に違反した医薬品を出荷していた。違反の発覚により、70品目を超える製品回収を行なった。

              厚生労働省が求める適切な安全管理は、①製造や品質試験等について手順を定め、②手順通りに作業を行い、③作業記録を作成すること である。しかし同社は作業手順があるにも関わらず 手順通りの検査が行われておらず 、その結果として不適合品が出荷されていた。
              また、不正発覚後、同社の主要製造拠点である富山工場は、富山県から約1ヶ月の営業停止命令を受けた。


              品質の問題が起きれば、たちまち企業の存続が危ぶまれる可能性がある。しかし、検査システムや検査体制を強化するほどに金銭的・人的負担が増し、日医工の事例のように 効率を優先して手順を守らない 人間が出てくる。


              # 失敗地点の廃棄コスト

              失敗が発生するポイントによって、その対応に必要となるコストが異なる。

              作業開始直後に異常を見つけた場合、原材料の廃棄で済む。一方、出荷の直前の発見では、各工程で投入した原材料だけでなく、作業員の工数も無駄になる。
              つまり、発見が遅れるほどコストは増大し、減ることはない。

              さらに、出荷後 に問題が起きれば、製品回収、クレーム対応、さらには先述の人命問題まで発展し得る。

              そのため、予防対策を行い、失敗を早期に検出する ための体制を整えることが重要である。


              # 品質コストの分類

              品質を保つためのアプローチは4種類ある。

              • 予防原価
                失敗の発生を防止するために掛かるコストである。
                例: 企画・設計のコスト、品質管理部門の人件費、従業員教育費

              • 評価原価
                仕入原材料、仕掛品・製品に異常がないか評価するコストである。
                例: 仕入材料の検査、出荷前の検査

              • 内部失敗原価 出荷前に失敗に気づき対処するコストである。
                例: 手直しの材料費・人件費、廃棄コスト

              • 外部失敗原価
                出荷してから失敗に気づき対処するコストである。
                例: 商品回収費用、クレーム対応の人件費、訴訟費用

              予防原価と評価原価は、失敗を発生させないための事前対策コストである。
              内部失敗原価と外部失敗原価は、失敗が起きた後の対応コストである。

              事前対策のコストと、対応コストはトレードオフの関係にある。 事前対策を強化すれば品質が高まり、対応コストは下がる。 事前対策が不十分なら失敗が増えて対応コストは増える。


              # 品質情報の活用事例

              項目名 内容
              参考URL 安全でおいしいお食事を (opens new window) / マクドナルド
              実施時期 2015年〜

              自社で製造する食品について、製造プロセス、原産国、衛生管理の取り組みといった 品質に係る情報を自社のウェブサイトで公開 している。
              商品の包装等にQRコードをつけ、スマートフォンから品質情報を確認することができるようにしている。 ウェブサイトでは、商品の安全性だけでなく、アレルギーや栄養に関する情報も参照できる。

              品質管理を徹底するだけでなく、生産管理体制や生産プロセスに関する情報を公開することで、消費者の安心を獲得し、リピーターの獲得に成功している。
