              # DXの定義

              # 経済産業省の定義

              デジタルトランスフォーメーション（DX）とは、デジタル技術を活用して事業を変革させ、顧客価値を創出することで、競争優位を確率することを指す。
              経済産業省や総務省の定義によると、デジタルトランスフォーメーション（DX）には4つの種類があるとされている。

              1. デジタイゼーション（Degitization）

                会社内の特定の工程における効率化のためにデジタルツールを導入するのが「デジタイゼーション」

              2. デジタライゼーション（Degitalization）

                自社内だけでなく外部環境やビジネス戦略も含めたプロセス全体をデジタル化するのが「デジタライゼーション」

              3. 狭い意味のデジタルトランスフォーメーション

                デジタル技術の活用による新たな商品・サービスの提供、新たなビジネスモデルの開発を通して、社会制度や組織文化なども変革していくような取組を指す

              4. 広い意味のデジタルトランスフォーメーション
                上記3種の全てをひとまとめに「デジタルトランスフォーメーション」と呼ばれる。


              # 1. デジタイゼーション

              デジタイゼーションは、現在の物理環境で行なっている業務をデジタル環境で行えるようにすることを指す。
              例えばペーパーレス活動が該当する。紙で行っていた作業をデジタル化することで検索性が高まり、物理的な保管スペースが不要になる。しかし、業務内容としては使うツールが紙からパソコンに変わるだけで業務内容が大きく変わらない。

              # 2. デジタライゼーション

              デジタライゼーションは、デジタル技術を使って現在の業務プロセスを変革する。

              例えば、AIを活用したデータ分析等が該当する。 熟練者が経験に基づく判断を行なっていた業務にAI支援を導入することで、若手でも熟練者と同等の判断をできる。 このようにデジタル技術を使った業務プロセスの変革がデジタライゼーションの主目的である。

              そして、英語読みではデジタリゼーションの発音が正しいと思う。

              # 3. 狭い意味のデジタルトランスフォーメーション

              狭い意味のデジタルトランスフォーメーション(狭義のDX)は、デジタル技術の活用によって新たな顧客価値を創り出す。
              例えば、食品配達のプラットフォームサービスが該当する。配達事業に参入して売上を伸ばしたいと考える飲食店と、行列に並ぶことなく食事をしたい一般消費者の両方のニーズに応え、 配達プラットフォームで両者をマッチングさせるシステムを運営している。外部に新しい顧客価値を提供することが狭義のDXの主目的である。

              # 4. 広い意味のデジタルトランスフォーメーション

              今まで書いた3種類を全部ひっくるめてDXとする。 IT技術を使っていさえすれば、なんでもDXなのである。
