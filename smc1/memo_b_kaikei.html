                                                                        # B.財務会計 MEMO


                                                                        財務・会計

                                                                        # 会計

                                                                        • アカウンティングという
                                                                        • 経営の実績をもとに書類を作成する

                                                                        # 財務会計

                                                                        • 損益計算書、貸借対照表、キャッシュフロー計算書などを用いて経営状態をステークホルダーに報告することを目的とする
                                                                        • 決まった処理方法、決まった表示形式で客観性を確保することが重要

                                                                        # 管理会計

                                                                        • 企業内部の意思決定社が適切な判断ができるようにすることを目的とする
                                                                        • 社内で使用するため、形式は自由。合理性が重要

                                                                        # 財務

                                                                        • ファイナンスという
                                                                        • 貨幣価値の観点から投資案の採択を行う
                                                                        • まだ発生していない将来の事象を定量的に評価する


                                                                        貸借対照表

                                                                        # 貸借対照表

                                                                        • その時の財政状況を表す
                                                                        • 資産の部、負債の部、純資産の部の3項目に分類される

                                                                        # 資産の部

                                                                        • 流動資産、固定資産、繰延資産に分類される

                                                                        # 流動資産

                                                                        • 正常営業循環基準、1yearルールで分類する

                                                                        # 固定資産

                                                                        • 有形固定資産、無形固定資産、投資その他の資産に分類される

                                                                        # 繰延資産

                                                                        • 創立費、開業費、社債発行費などの勘定項目が含まれる

                                                                        # 負債の部

                                                                        • 流動負債、固定負債に分類される

                                                                        # 純資産の部

                                                                        # 貸借対照表の表示方法

                                                                        • 流動性配列法と固定性配列法がある。項目の並び順が異なる

                                                                        # 流動性配列法

                                                                        • 流動資産、流動負債を上に表示する
                                                                        • 企業の支払い能力を把握しやすい
                                                                        • 一般企業の多くがこちらを採用している

                                                                        # 固定性配列法

                                                                        • 固定資産、固定負債を上に表示する
                                                                        • 純資産を負債より上に表示する方法もある
                                                                        • 固定資産が長期資産でどのように賄われているかを把握しやすい
                                                                        • 固定資産の多いインフラ企業で採用されることがある


                                                                        資産

                                                                        # 流動資産の条件

                                                                        # 正常営業循環基準

                                                                        • 正常営業循環基準は、企業の営業を行う上で循環する資産・負債を流動資産・流動負債に記載するルール
                                                                        • 例えば、原材料の仕入、加工、在庫、売上、売掛金、現金というプロセスを対象とする

                                                                        # 1yearルール

                                                                        • 1年以内に現金化、費用化する資産を流動資産として扱うルール

                                                                        # 現金及び預金

                                                                        • 現金、普通預金、用座預金、定期預金などのこと

                                                                        # 債権の分類

                                                                        • 営業取引で発生した債権は、受取手形、売掛金などに計上する
                                                                        • 営業外の取引で発生した債権は、貸付金、未収入金などに計上する

                                                                        # 売掛金

                                                                        • 商品を売ってもすぐに回収するのではなく、後日払う約束をすること
                                                                        • 企業間信用取引とも呼ばれる

                                                                        # 前渡金、前払金

                                                                        • 商品売買の手付金のようなもの

                                                                        # 未収金、未入金

                                                                        • 営業外の土地などを売った時に発生する債権
                                                                        • 営業上の売買は売掛金

                                                                        # 手形

                                                                        # 為替手形

                                                                        • A社に対する売掛金と、B社に対する買掛金があるとき、現金を受け取って支払う手間が掛かる。そこで、両社に対して為替手形を発行し、両社間で現金のやりとりをしてもらう
                                                                        • 両社では、受取手形、支払手形で処理する
                                                                        • 手形の金額を払う人を、支払人、名宛人または引受人という
                                                                        • 手形の金額を受け取る人を、受取人または指図人という

                                                                        # 裏書譲渡

                                                                        • 受取手形を他の人に譲渡して換金すること
                                                                        • 手形には決済期日があり、早期決済分が割り引かれる(手形割引)

                                                                        # 不渡手形

                                                                        • 支払手形を期日までに支払えないと、不渡手形になる。
                                                                        • 半年間で2度の不渡を出すと、銀行との取引ができなくなり、事実上の倒産となる

                                                                        # 偶発債務

                                                                        • 取引先の倒産など、一定確率で発生する債務(支払う義務)を偶発債務という
                                                                        • 取引先倒産の理由とはいえ、支払い義務は逃れられない

                                                                        # 外貨建の資産

                                                                        • ドル、ユーロなど
                                                                        • 決算日に決算日時点のレートで換算する
                                                                        • 換算による差額は、為替差損益として、P/Lに表示される
                                                                        1. 1$=100円で1,000ドルを持っている場合、1,000ドル×90円のため10万円である
                                                                        2. 決算日に1$=90円だった場合、1,000ドル×90円で9万円となる
                                                                        3. 10万円の価値だったものが9万円に目減してるため、差額の1万円を為替差損益に計上する

                                                                        # 有価証券

                                                                        • 保有目的によって勘定科目が異なる
                                                                        • 取得にかかった手数料を含めて、有価証券として資産計上する

                                                                        # 売買目的有価証券

                                                                        • 売買によってキャピタルゲインを得ることが目的の有価証券(株式、債権等)である
                                                                        • いつでも売却できることから、流動資産に分類される
                                                                        • 売却損益は、営業外損益として処理される

                                                                        # 満期保有目的債権

                                                                        • 満期まで保有して利息や額面を受け取ることが目的の有価証券(社債、国債等)をいう
                                                                        • 固定資産に分類される
                                                                        • 売却損益は、特別損益として処理される

                                                                        # 子会社株式、関連会社株式

                                                                        • 企業を支配することが目的の株式をいう
                                                                        • 保有する割合によって勘定科目が変わる
                                                                        • 固定資産に分類される
                                                                        • 売却損益は、特別損益として処理される

                                                                        # その他有価証券

                                                                        • 上記のいずれの目的でもない有価証券をいう
                                                                        • 例えば、業務提携などの持合株式など
                                                                        • 固定資産に分類される
                                                                        • 売却損益は、特別損益として処理される

                                                                        # 有価証券の評価

                                                                        • 有価証券は、期末に金額の評価を行う
                                                                        • 売買目的有価証券
                                                                          • 時価評価を行い、差額を当期損益に計上する
                                                                        • 満期保有目的、子会社株式、関連会社株式
                                                                          • 取得価額のまま
                                                                        • その他有価証券
                                                                          • 純資産の部に計上する

                                                                        # 固定資産

                                                                        # 有形固定資産

                                                                        • 長期にわたって営業に使用する資産をいう
                                                                        • パソコン、製造設備、社用車など
                                                                        • 減価償却によって使用した分を徐々に減額する
                                                                        • 取得にかかった手数料、搬入費用、整地費用などを含めた金額を、固定資産として計上する

                                                                        # 無形固定資産

                                                                        • ソフトウェア、特許権、商標権など

                                                                        # 投資その他資産

                                                                        • 上記以外の固定資産をいう
                                                                        • 子会社株式などの有価証券、貸付金など

                                                                        # 減損処理

                                                                        • 固定資産が帳簿価額以上のキャッシュを生まないと判断される時、帳簿価額を減らし、減損処理を行う
                                                                          • 事業環境の変化、設備の陳腐化など
                                                                        • 減損した分は、特別損失として処理する

                                                                        # 減損兆候の認識

                                                                        • 帳簿価額と、将来キャッシュフローの現在価値を比較する
                                                                        • 帳簿価額よりキャッシュフローが少なければ、減損処理を開始する

                                                                        # 減損損失額の算出

                                                                        • 正味売却価値(いま売却した時に得られる金額)と、使用価値(将来キャッシュフロー現在価値)を比較する
                                                                          • 金額の多い方が、その資産の価値である
                                                                        • その金額まで帳簿価額を減らす

                                                                        # 計算例

                                                                        • 簿価(5,000万円)
                                                                        • 正味売却価値(4,000万円) = 時価(4,500万円) - 処分費用(500万円)
                                                                        • 使用価値(3,950万円) = 各期キャッシュフロー(1,000万円) × 4年の割引(3.9505)
                                                                          • 正味売却価値の方が高いので、そこまで金額を減らす
                                                                        • 減損損失額(500万円) = 簿価(5,000万円) - 正味売却価値(4,000万円)

                                                                        # 繰延資産

                                                                        投資の効果が数年に渡って発生する場合、一時的に資産計上しておき、毎年費用化していく

                                                                        • ◯◯費と書いているが、費用ではなく資産計上する。

                                                                        # 創立費

                                                                        • 会社を立ち上げる際にかかった費用をいう
                                                                        • 会社登記の免許税や、定款作成の費用をいう
                                                                        • 設立から5年以内に償却する

                                                                        # 開業費

                                                                        • 会社の設立から営業開始までにかかった費用をいう
                                                                        • 名刺作成費用、 広告費用等の費用をいう
                                                                        • 開業から5年以内に償却する

                                                                        # 株式交付費

                                                                        • 株式を関連手続きにかかった費用
                                                                        • 新株発行、自己株式の処分等の費用をいう
                                                                        • 株式交付から3年以内に償却する

                                                                        # 開発費

                                                                        • 技術開発や、市場開拓にかかった費用をいう
                                                                        • 5年以内に焼却する

                                                                        # 社債発行費等

                                                                        • 利息法で月割償却する


                                                                        負債

                                                                        # 負債

                                                                        • 負債とは、金銭を支払わなくてはならない債務のことをいう

                                                                        # 流動負債

                                                                        # 支払手形

                                                                        # 買掛金

                                                                        # 短期借入金

                                                                        • 返済期限が1年以内の借入金をいう
                                                                        • 金融機関(銀行など)から借りたお金をいう

                                                                        # 未払金

                                                                        • 営業外の取引で生じた債務である
                                                                        • 有価証券や土地を購入したことで発生したもの
                                                                        • 支払い期日が1年より先のものは、長期未払金として固定負債に表示する

                                                                        # 前受金

                                                                        # 預り金

                                                                        • 源泉徴収分を一時的に預かっている状態をいう

                                                                        # 固定負債

                                                                        # 社債

                                                                        • 市場から資金を調達するため、会社として発行した債券のこと
                                                                        • 社債の発行
                                                                          • 社債は、債券に記載された金額(額面)とは異なる金額(払込価額)で購入を募ることができる
                                                                          • 額面と払込価額の差は、償却原価法によって毎年額面に近づける調整処理を行う
                                                                        • 社債の運用
                                                                          • 社債は定期的に利息を払う必要がある
                                                                          • 利息は額面×利率を払う
                                                                        • 社債の償還
                                                                          • 満期を迎えたら、債券と額面(債券に記載された金額)の現金を交換する。これを償還という

                                                                        # 長期未払金

                                                                        • 未払金(営業外の取引で生じた債務)の支払い期日が1年より長いもの

                                                                        # 退職給付引当金

                                                                        • 退職金制度を用意している場合、退職時期に向けて毎年引当金として費用計上し、累計額を引当金計上する
                                                                        • 退職金の支払い時に取り崩す


                                                                        純資産

                                                                        # 純資産

                                                                        • 企業は株主の持ち物で、社長は経営を任されている関係にある。
                                                                        • 株主から調達した資金や、企業経営によって蓄積した剰余金が純資産であるため、自己資本とも呼ばれる。
                                                                        • 返済の義務がない。

                                                                        # 資本金

                                                                        • 会社の財産を確保するための基準となる金額
                                                                        • 払込額の1/2までを資本剰余金とすることができる
                                                                          • 全額を資本金とすることもできる
                                                                          • 資本金が少ないと中小企業向けの政策を受けられるメリットがある
                                                                          • 資本金が多いと取引先からの信用が得られるメリットがある
                                                                        • 公開会社は、会や設立の際に、発行可能株式の1/4を株式として発行する必要がある

                                                                        # 利益剰余金

                                                                        • 企業が獲得した利益の積み重ねである
                                                                        • 利益準備金とその他利益剰余金で構成される

                                                                        # 資本剰余金

                                                                        • 利益以外を源泉とする剰余金である
                                                                        • 資本準備金とその他資本剰余金で構成される

                                                                        # 自己株式

                                                                        • 市場に発行した株式を自ら取得したものである
                                                                        • 純資産の部でマイナス表記される
                                                                        • 自己株式を取得するメリット
                                                                          • 買収の対価として使用できる
                                                                          • 敵対的買収を防ぐ
                                                                          • 市場に流通する株式数が減ることで、一株当たりの価値が上昇する
                                                                        • 取得手続き
                                                                          • 通常は株主総会の通常決議
                                                                          • 特定株主から買い取る場合は、特別決議
                                                                        • 消却(※償却ではない)
                                                                          • 株式をなくすこと

                                                                        # その他有価証券評価差額金

                                                                        • 持ち合い株式(提携企業間でお互いに株を持つ等)の簿価と時価の差額
                                                                          • 持ち合い株式は、簿価と評価額の差額を毎期算出する
                                                                          • 算出した差額は、損益計算書ではなく、貸借対照表(純資産)で表示される

                                                                        # 新株予約権

                                                                        • 将来、自己株式を一定の金額で受ける権利のこと
                                                                        • 企業側は、払込予定金額を計上する

                                                                        # 配当の手続き

                                                                        • 企業が得た利益を、株主に対して現金として利益還元する制度を配当という
                                                                        • 損益計算書の税引後純利益から、配当金を引いた額が、貸借対照表の利益剰余金になる

                                                                        # 配当の積み立て

                                                                        • 配当金の一部は積み立てなければならない。以下いずれかの少ない金額を積み立てる
                                                                          1. 配当金額の1/10
                                                                          2. 資本金 × 1/4 - (資本準備金＋利益準備金)
                                                                        • 資本準備金と利益準備金の合計が、資本金の1/4を超えていれば、積み立ては不要となる。
                                                                        • 配当の原資は、その他資本剰余金か、その他利益剰余金である。
                                                                          • その他資本剰余金から配当をする場合は、資本準備金に積み立てる
                                                                          • その他利益剰余金から配当をする場合は、利益準備金に積み立てる

                                                                        # 分配可能額

                                                                        • 配当または自己株式の取得を行う場合、その簿価に上限が設定されている。これを分配可能額という
                                                                        • 純資産総額から、資本金と準備金を差し引いた残額である
                                                                        • 配当によって純資産が300万円を下回る場合、配当してはならない


                                                                        損益計算書

                                                                        # 損益計算書

                                                                        • 企業の経営成績を表す
                                                                        • 純利益から配当金が出されるので、配当可能額を算出する目的もある
                                                                        • 利益 = 収益 - 費用 の関係にある
                                                                        • 利益を出すためには、収益を増やすか、費用を減らす

                                                                        # 損益計算書の構造

                                                                        # 営業損益の部

                                                                        • 売上高総利益(粗利) = 売上高 - 売上原価
                                                                        • 営業利益 = 売上高総利益 - 販売費及び一般管理費 である

                                                                        # 営業外損益の部

                                                                        • 経常利益 = 営業利益 - 営業外収支 である

                                                                        # 特別損益の部

                                                                        • 税引前当期純利益 = 経常利益 + 特別利益 - 特別損失 である

                                                                        # 収益

                                                                        • 収益には、売上高の他に営業外の受取利息、受取配当金、固定資産売却益などが含まれる

                                                                        # 売上高

                                                                        • 販売目的の商品やサービスについて、当期販売した総額を表す。
                                                                        • 以下の4項目は売上高から控除する
                                                                          • 売上戻り
                                                                            • 返品のこと
                                                                          • 売上値引
                                                                            • 販売した後に値引きをすること
                                                                              • 不良品や破損等
                                                                          • 売上割戻
                                                                            • 大量販売によるキャッシュバックのこと
                                                                            • 代理店が販売ノルマ達成した時など
                                                                          • 売上割戻引当金繰入
                                                                            • キャッシュバックに備えた引当金
                                                                            • 来期発生する見込み額を見積もって計上する

                                                                        # 販売費及び一般管理費

                                                                        • 販売費は、販売業務に関連して発生した費用をいう
                                                                        • 一般管理費は、全般的な管理業務で発生した費用をいう
                                                                        • 省略して、販管費という
                                                                        • 販管費には以下のようなものが含まれる
                                                                          • 商品運搬の費用
                                                                          • コマーシャルなどの広告宣伝費用
                                                                          • 社会保険などの福利厚生費用
                                                                          • 本社社員の給与、役員給与
                                                                          • 水道光熱費
                                                                          • 固定資産などの減価償却費

                                                                        # 営業外収支

                                                                        • 企業が主とする営業活動ではない収入と支出を表現する
                                                                        • 営業外費用には、以下のようなものが含まれる
                                                                          • 借入金から生じる支払利息
                                                                          • 社債から生じる社債利息
                                                                          • 売上割引
                                                                            • 売掛金が早期に入金された時、一部(利息分)を減額すること
                                                                            • 早期入金によって企業経営が安定する
                                                                            • 商品の売買ではなく、決済に関する事項なので営業外となる
                                                                          • 有価証券売却損
                                                                        • 営業外収益には、以下のようなものが含まれる
                                                                          • 預金から生じる受取利息
                                                                          • 保有する社債による有価証券利息
                                                                          • 保有する株式による受取配当金
                                                                          • 仕入割引
                                                                          • 有価証券売却益

                                                                        # 特別損益

                                                                        • 通常の営業活動において毎期連続で発生しないような損益をいう
                                                                          • 固定資産の売却、災害損失など
                                                                        • 固定資産の売却は、簿価と売価の差額が計上される
                                                                          • 差額がプラスの場合、固定資産売却益
                                                                          • 差額がマイナスの場合、固定資産売却損

                                                                        # 収益や費用の計上時期

                                                                        • 発生時期と、現金の支払い時期にずれが生じる時、どのタイミングで計上するか

                                                                        # 発生主義(費用計上の決まり)

                                                                        • 会計期間内に発生した費用を、当期の費用として処理する考え方
                                                                        • 例えば、4Qごとに損益計算書を作っている場合、1年分の利息を先に支払っても1Q分の費用を計上する

                                                                        # 実現主義(収益計上の決まり)

                                                                        • 現金、手形、売掛金などの貨幣性資産を受け取った時に収益とする考え方

                                                                        # 現金主義

                                                                        • 現金を扱った時に費用や収益を計上する考え方

                                                                        # 工事収益基準

                                                                        • 長期の請負工事に関する収益は、進捗度に応じた計上を行う工事進行基準による処理が許可されている

                                                                        # 工事完成基準

                                                                        • 工事が完成した時に計上する考え方


                                                                        記帳の流れ

                                                                        # 記帳の流れ

                                                                        • 取引を行った結果は、以下の順番に処理していく
                                                                          • 取引の実行
                                                                          • 仕訳
                                                                          • 総勘定元帳へ記載
                                                                          • 合計残高試算表を作成
                                                                          • 精算表を作成
                                                                          • 財務諸表を作成

                                                                        # 主要簿

                                                                        • 主要簿は、どの企業の必ず備えなければならない
                                                                        • 仕訳帳と総勘定元帳のこと

                                                                        # 仕訳帳

                                                                        • 普通仕訳帳と、特殊仕訳帳がある

                                                                        # 普通仕訳帳

                                                                        • 全ての取引を、取引のあった順に記録する帳簿である
                                                                        • 1つの仕訳帳に全ての取引を記帳して、元帳に転記する方法を単一仕訳帳制という

                                                                        # 特殊仕訳帳

                                                                        • 頻繁に発生する特定取引だけを記入する仕訳帳である
                                                                        • 補助記入帳を仕訳帳として使用する
                                                                        • 特殊仕訳帳制という

                                                                        # 総勘定元帳

                                                                        • どの勘定科目がどれだけ損減したかを把握するため、金額を集計する帳簿のこと
                                                                        • 総勘定元帳は、企業で使用する勘定項目が全て網羅されている

                                                                        # 補助簿

                                                                        • 必要に応じて備える帳簿
                                                                        • 補助記入帳と、補助元帳がある

                                                                        # 補助記入帳

                                                                        • 取引が発生した順に内容を記録する帳簿
                                                                          • 現金出納帳
                                                                          • 当座預金出納帳
                                                                          • 売上帳
                                                                          • 仕入帳
                                                                          • 受取手形記入帳
                                                                          • 仕入手形記入帳　など

                                                                        # 補助元帳

                                                                        • 取引相手や内容で区別して記録する帳簿
                                                                          • 売掛金元帳
                                                                          • 買掛金元帳
                                                                          • 商品有高帳　など

                                                                        # 転記

                                                                        • 仕訳帳から総勘定元帳に書き写すこと
                                                                        • 以下の手順で行う
                                                                          • 特殊仕訳帳から合計を転記する
                                                                          • 特殊仕訳帳の諸口は個別転記をする
                                                                          • 普通仕訳帳を個別転記をする

                                                                        # 伝票式会計

                                                                        • 仕訳帳の代わりに、伝票を使って会計を記録する方法である
                                                                        • 伝票を総勘定元帳に転記する
                                                                        • 伝票の種類
                                                                          • 入金伝票
                                                                          • 出金伝票
                                                                          • 仕入伝票
                                                                          • 売上伝票
                                                                          • 振替伝票
                                                                        • ３伝票制では、入金、出金、振替の伝票だけを使う
                                                                        • ５伝票制では、入金、出金、振替、売上、仕入の伝票を使う

                                                                        # 合計残高試算表

                                                                        • 総勘定元帳の記入が正しく行われたかを確認する表
                                                                        • 決算整理事項の金額を調整して、当期収益・当期費用を適正化する
                                                                        • 決算整理事項とは、例えば売上原価の算出、減価償却費の算出、引当金の算出、など

                                                                        # 試算表

                                                                        • ミスの防止が目的である
                                                                        • 貸方と借方が一致すれば、ミスの可能性は低いと考えられる
                                                                        • 試算表は3種類ある
                                                                          • 合計試算表
                                                                            • 各勘定における貸方の合計と、借方の合計を比較して、記録の確認を行う
                                                                          • 残高試算表
                                                                            • 各環境における貸方残高と、借方残高を集計する表である
                                                                          • 合計残高試算表
                                                                            • 合計試算表と残高試算表の機能をひとまとめにした表である

                                                                        # 精算表

                                                                        • 決算の見通しを立てるために作成する表


                                                                        決算整理

                                                                        # 決算整理

                                                                        • 企業は一定の期間ごとに、営業成績を財務諸表で報告する義務がある

                                                                        # 決算の流れ

                                                                        • 取引を受けて、仕訳をする
                                                                        • 総勘定伝票に転記する
                                                                        • 試算表を作成する
                                                                        • 精算表を作成する
                                                                        • 財務諸表を作成する

                                                                        # 売上原価(三分法)

                                                                        • さんぶんほう、と読む
                                                                        • 製品化して販売した分が売上原価として計上される
                                                                        • 売上原価は①期首在庫、②当期仕入、③期末在庫から求める
                                                                        • 売上原価 = 期首在庫 + 当期仕入 - 期末在庫
                                                                          • 箱図を作ると分かりやすい

                                                                        # 棚卸資産の評価

                                                                        # 先入先出法(FIFO)

                                                                        • First in, First out
                                                                        • 先に仕入れた製品から順に売れていったと仮定する計算方法である
                                                                        • 計算手順
                                                                          • 当期仕入の1個当たり金額を算出する-①
                                                                          • 期末在庫に-①を掛ける
                                                                          • 残りが売上原価になったものと考える

                                                                        # 移動平均法

                                                                        • 商品を仕入れるたびに移動平均を算出し、その平均額で売れていったと考える計算方法である

                                                                        # 棚卸資産減耗損

                                                                        • 期末に商品の数(及び評価額)を数え、紛失や破損などがあれば、簿価との差額を損失計上する
                                                                        • 棚卸資産減耗損は、売上原価として計上される
                                                                          • 販管費として計上することも可能
                                                                        • 金額の際は、価格差異と数量差異がある。
                                                                          • T字を書くとわかりやすい
                                                                          • 左下から「ひじ」、「ひじ」

                                                                        # 価格差異

                                                                        • (標準価格 - 実価格) × 実価格

                                                                        # 数量差異

                                                                        • 標準価格 × (標準価格 - 実価格)

                                                                        # 減価償却費

                                                                        • 建物、機械などの固定資産は、未来永劫使える訳ではなく、使用した分だけ価値が減少する
                                                                        • その期に資産価値が減少した分を減価償却費として費用化することで、収益と費用を対応させる

                                                                        # 法定耐用年数

                                                                        • 各企業が自由に償却しては決算数値を操作できてしまうので、法律で耐用年数を定めている
                                                                        • 耐用年数の間、一定のルールで価値が減少していく

                                                                        # 定額法

                                                                        • 毎期、同額を償却する方法である
                                                                        • 減価償却費 = 取得価額 × 償却率
                                                                        • 定額法の償却率は、1÷耐用年数で算出できる

                                                                        # 定率法

                                                                        • 毎期、残額に償却率を掛けて費用を算出する方法である
                                                                        • 減価償却費 = (取得価額 - 償却済金額) × 償却率
                                                                        • 定率法とは償却率は異なる。

                                                                        # 200%定率法

                                                                        • 定額法で算出した償却率の2倍(200%)の償却率を使い、定率法で償却する方法である。
                                                                        • 一定水準以上まで償却をした場合、残額を残年数の定額法で処理する

                                                                        # 生産高比例法

                                                                        • 生産可能数を前提として、生産した数の分だけ償却を行う方法である
                                                                        • 取得価額 × (当期生産量 ÷ 総生産可能量)

                                                                        # 貸倒引当金

                                                                        • 取引先の倒産などにより、売掛金などを回収できなくなることを貸倒という
                                                                        • あらかじめ来期に発生する貸倒の可能性を見積もって、当期費用として貸倒引当金を計上することができる
                                                                        • 貸倒引当金が未設定にも関わらず貸倒が発生した場合は、貸倒損失として計上する

                                                                        # 貸倒引当金の計上条件

                                                                        • 以下の要件を全て満たす場合に限り、引当金計上が可能である
                                                                          • 将来発生する貸倒に対する費用であること
                                                                          • 当期(あるいは当期以前)に発生した債権に関する貸倒であること
                                                                          • 過去の発生状況などから、発生可能性や金額を合理的に見積もることができること
