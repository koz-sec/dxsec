                                                                        生産管理

                                                                        # 生産方式

                                                                        # Just in time

                                                                        • 「すべての工程が、後工程の要求に合わせて、必要な物を、必要なときに、必要な量だけ生産(供給)する生産方式。

                                                                        # オーダーエントリー方式

                                                                        • 生産工程にある製品に顧客のオーダーを引き当てて、製品の仕様の選択または変更をする生産方式

                                                                        # 生産座席予約方式

                                                                        • 受注時に、製造設備の使用日程・資材の使用予定などにオーダーを割り付け、顧客が要求する納期どおりに生産する方式

                                                                        # 整番管理方式

                                                                        • 製造命令書を発行するときに、その製品に関するすべての加工と組立の指示書を準備し、同一の製造番号をそれぞれに付けて管理を行う方式

                                                                        # 追番管理方式

                                                                        • 生産の計画と実績に追番(累計番号)を付け、計画と実績の差を把握することで、その製造実績の進捗状況を管理すること

                                                                        # 管理指標

                                                                        # 不適合品率

                                                                        • 不適合品の数 ÷ 検査した総数で求める。割合が少ないほど良い。
                                                                        • 例) 200件を検査して、5件が不適合だった
                                                                        • 5 ÷ 200 = 0.025 (= 2.5%)

                                                                        # 歩留り

                                                                        • 産出量 ÷ 投入量で求める。割合が多いほど良い。
                                                                        • 例) 200個を投入して、190個が産出された。
                                                                        • 190 ÷ 200 = 0.95 (= 95%)

                                                                        # 作業能率

                                                                        • 標準工数 ÷ 実績工数で求める。割合が多いほど良い。
                                                                        • 例) 標準では120時間かかる作業を、実績で100時間で終えた。
                                                                        • 120 ÷ 100 = 1.2 (= 120%)

                                                                        # 労働生産性

                                                                        生産量 ÷ 労働量で求める。


                                                                        # 在庫管理

                                                                        # ダブルビン方式

                                                                        1つの在庫に2つの置き場所を用意し、一方が空になったら満タンにするよう固定量の発注を行う方式である。

                                                                        # 経済的発注点

                                                                        EOQ: Economic Order Quantity)は、発注コストと在庫コストがトレードオフの関係であると仮定して、コストが最小限となる発注量のことである。

                                                                        # 発注点

                                                                        発注の必要なタイミングのことで、通常、残り在庫量を指す。

                                                                        # 作業標準

                                                                        作業標準は、誰がやっても結果が出せるように定めるものである。

                                                                        • 安全・品質・効率を達成する手段である。
                                                                        • 写真、イラスト、動画等で補完されると理解しやすい。
                                                                        • 無理なく、安全にできることが重要で、変化に合わせた改廃が必要である。

                                                                        # CRP

                                                                        POSデータから、購入された分だけ補充する仕組み

                                                                        # VMI(vender managed inventory)

                                                                        納入を行う業者に在庫管理も委託すること

                                                                        # JTRN(ジェイトラン)

                                                                        • 国内の物流EDIの標準規格
                                                                        • Japan Transportの略と覚えて良い
                                                                        • 参考: 日本通運 (opens new window)

                                                                        # ASN(Advanced Shipping Notice)

                                                                        • 納入業者が物流センター側に出荷情報を事前に通知する
                                                                        • 「事前出荷情報」と呼ばれる
                                                                        • 参考:日本通運 (opens new window)

                                                                        • 商品ごとに売上を棒グラフで表現すると、上位2割から3割の商品が売上の大半を占める
